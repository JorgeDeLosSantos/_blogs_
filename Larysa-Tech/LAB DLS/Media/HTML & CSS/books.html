Libros y Apuntes

En esta sección se muestran algunos libros acerca de diversos lenguajes de programación.

Python

  • Python no muerde, yo sí. (Roberto Alsina)
  • Python para todos. (Raúl González Duque)
  • El tutorial de Python. (Guido Van Rossum) [Traducido por Python Argentina]
  • Introducción a la programación con Python. (Andrés Marzal & Isabel Gracia)
  • Introduction to Python for Econometrics, Statistics and Data Analysis. (Kevin Sheppard)
  • Making games with Python and Pygame. (Al Sweigart)
  • Introduction to Scientific Computing in Python. (Robert Johansson)
  • A Primer on Scientific Programming with Python. (Hans Petter Langtangen)

MATLAB

  • Programación en MATLAB, fundamentos y aplicaciones. (Jorge De Los Santos)
  • Aprenda MATLAB como si estuviera en primero. (Javier García de Jalón, et al.)
  • MATLAB a través de ejemplos. (César Pérez López)
  • Análisis estructural con MATLAB. (José A. Inaudi & Juan C. de Llera)
  • Prácticas de Análisis Matricial Aplicado y Ampliación de Métodos Numéricos con MATLAB. (Ion Zavala)
  • An Introduction to Digital Image Processing with MATLAB. (Alasdair McAndrew)
  • Introduction to Programming in MATLAB for Chemical and Biomolecular Engineering Students. (Sam H. Davis)
  • Prácticas de Análisis Matricial Aplicado y Ampliación de Métodos Numéricos con MATLAB. (Ion Zavala)
